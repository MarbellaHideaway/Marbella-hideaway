# Marbella Hideaway Next.js Website

A premium rebuild of the Marbella Hideaway website using Next.js and Vercel.

## Deploy

Upload the contents of this folder to the `MarbellaHideaway/Marbella-hideaway` GitHub repository root.

Vercel should automatically detect it as a Next.js project.

## Local development

```bash
npm install
npm run dev
```
